How it works

01

Read

Claude vision transcribes the printed fields, word-for-word. It only reads — it never guesses missing text.

02

Match

Brand and ABV are matched with judgment: case, punctuation, and rounding are forgiven; real mismatches are flagged.

03

Enforce

The Government Warning is held to the exact 27 CFR 16.21 wording, capitalized and bold. Anything off is rejected.

04

Return

A plain pass / needs-review / fail per field, in about three seconds, with the exact reason.

Why these choices. The model extracts; deterministic code decides. That keeps every verdict explainable and consistent, instead of asking an AI to "grade" a label differently each run. Speed was a hard requirement — the prior scanning vendor lost the room at 30–40 seconds, so this uses a fast model and ships one label per request, which also lets a 300-label dump screen concurrently rather than in a queue.

On the federal network. The model call is one outbound HTTPS request to a single endpoint, made server-side — not from the agent's browser — so it sits behind one allow-list entry rather than the scattered ML endpoints that broke the last pilot. Full source, setup, assumptions, and trade-offs are in the repository.
